# CapSure - 轻量级屏幕录制工具

一个简洁、高效的桌面屏幕录制软件，支持全屏录制、系统声音+麦克风双轨采集、全局快捷键控制。

## 功能特性

### 录制功能
- **全屏录制** — 支持 1080p / 720p / 4K 分辨率
- **帧率选择** — 24 / 30 / 60 FPS
- **多格式输出** — MP4 / MKV / WebM
- **双轨音频** — 系统声音（Loopback）和麦克风独立采集、实时混音
- **独立音量控制** — 系统声音和麦克风音量分别调节
- **暂停/继续** — 录制中随时暂停和恢复

### 操控体验
- **全局快捷键** — `Ctrl+Shift+R` 录制、`Ctrl+Shift+P` 暂停、`Ctrl+Shift+S` 停止
- **倒计时** — 录制前 3 秒倒计时提示
- **系统托盘** — 最小化到托盘，后台运行不打扰
- **录制时置顶** — 录制中窗口自动置顶，方便观察状态
- **实时预览** — 录制画面缩略图实时预览
- **环形进度** — 录制按钮显示环形进度动画

### 界面设计
- **暗色主题** — 舒适的深色界面
- **紧凑窗口** — 小巧（300×260），不占空间
- **设置持久化** — 所有设置自动保存，重启不丢失
- **按钮 Tooltip** — 悬停显示功能提示

### 数据安全
- **崩溃保护** — 程序异常退出时自动保存已录制内容
- **录制历史** — 自动记录所有录制历史（时长、大小、格式）

## 快速开始

### 环境要求
- Python 3.11+
- Windows 10 / 11

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行程序
```bash
python main.py
```

### 打包为可执行文件
```bash
python scripts/build.py
```
打包完成后，可执行文件位于 `dist/CapSure.exe`

## 系统声音录制说明

录制系统声音需要在 Windows 声音设置中启用「立体声混音」：

1. 右键任务栏喇叭 → **声音设置**
2. **更多声音设置** → **录制** 选项卡
3. 右键空白处 → **显示禁用的设备**
4. 找到 **立体声混音 (Stereo Mix)** → 右键启用

如无此选项，可安装虚拟音频驱动（如 [VB-Cable](https://vb-audio.com/Cable/)）。
