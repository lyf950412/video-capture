# CapSure - 轻量级屏幕录制工具

一个简洁、高效的桌面屏幕录制软件，支持视频录制、音频采集、自动保存等功能。

## 功能特性

### 核心功能
- **屏幕录制** - 全屏录制，支持 1080p/720p/4K 分辨率
- **帧率选择** - 支持 24/30/60 FPS
- **多格式输出** - MP4、MKV、WebM 格式
- **音频采集** - 支持麦克风和系统音频录制
- **音量控制** - 独立调节麦克风和系统音频音量
- **自动设备选择** - 智能选择最佳音频输入设备

### 录制控制
- **开始/停止** - 一键开始和停止录制
- **暂停/继续** - 录制过程中可随时暂停和恢复
- **实时计时** - 显示录制时长
- **文件大小** - 实时显示录制文件大小

### 数据安全
- **崩溃保护** - 程序崩溃时自动保存已录制内容
- **录制历史** - 自动记录所有录制历史
- **自动保存** - 停止录制后自动保存到指定目录

### 界面设计
- **现代暗色主题** - 舒适的深色界面，减少视觉疲劳
- **紧凑布局** - 小巧窗口，不占用过多屏幕空间
- **流畅动画** - 录制按钮脉冲动画效果
- **设置面板** - 独立设置窗口，配置录制参数

## 快速开始

### 环境要求
- Python 3.11+
- Windows 10/11

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行程序
```bash
python main.py
```

### 打包为可执行文件
```bash
python scripts/build.py
```
打包完成后，可执行文件位于 `dist/CapSure.exe`

## 项目结构

```
video-record/
├── main.py                 # 程序入口
├── requirements.txt        # 依赖列表
├── assets/
│   └── icon.ico           # 应用图标
├── scripts/
│   ├── build.py           # 打包脚本
│   └── generate_icon.py   # 图标生成脚本
├── src/
│   └── core/

│       └── history.py     # 录制历史
├── ui/
    └── main_window.py     # 主界面
```

## 待改进事项

### 高优先级
1. **系统音频录制** - 当前系统无 WASAPI Loopback 设备，无法直接录制系统播放的声音。建议用户安装虚拟音频设备（如 VB-Cable）或启用立体声混音
2. **区域录制** - 目前仅支持全屏录制，需要添加选择录制区域的功能
3. **摄像头画中画** - 添加摄像头画面叠加功能

### 中优先级
4. **快捷键支持** - 添加全局快捷键控制录制（开始/停止/暂停）
5. **鼠标高亮** - 录制时高亮显示鼠标点击位置
6. **录制预览** - 实时预览录制画面
7. **多显示器支持** - 选择特定显示器进行录制
8. **定时录制** - 支持定时开始和自动停止

### 低优先级
9. **视频编辑** - 简单的视频剪辑功能
10. **GIF 导出** - 支持导出为 GIF 格式
11. **水印功能** - 添加自定义水印
12. **云端同步** - 录制历史云端同步
13. **多语言支持** - 国际化支持

### 已知问题
- 长时间录制后音频合并可能较慢（已改为后台异步处理）
- Windows 音量合成器中应用音量可能被系统锁定
- 部分音频设备可能无法被正确识别
